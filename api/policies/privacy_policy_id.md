# Kebijakan Privasi
**Versi 1.0**  
**Tanggal Efektif: 1 Januari 2024**

## 1. Pendahuluan
Lumina Technologies ("kami", "perusahaan", atau "Lumina OS") berkomitmen untuk melindungi privasi dan data pribadi Anda sesuai dengan Undang-Undang Perlindungan Data Pribadi (UU PDP) Indonesia.

## 2. Data yang Kami Kumpulkan
Kami mengumpulkan data pribadi berikut:
- **Data Identitas**: Nama, email, nomor telepon
- **Data Bisnis**: Nama perusahaan, lokasi, informasi kontak
- **Data Penggunaan**: Log aktivitas, preferensi pengguna
- **Data Teknis**: Alamat IP, jenis perangkat, browser

## 3. Dasar Hukum Pemrosesan
Kami memproses data Anda berdasarkan:
- **Persetujuan**: Untuk pemasaran dan personalisasi
- **Kontrak**: Untuk penyediaan layanan
- **Kepentingan Sah**: Untuk analisis dan keamanan

## 4. Hak Anda
Sesuai UU PDP, Anda memiliki hak:
- **Hak Akses**: Mengakses data pribadi Anda
- **Hak Koreksi**: Memperbaiki data yang tidak akurat
- **Hak Penghapusan**: Meminta penghapusan data
- **Hak Portabilitas**: Menerima data dalam format terbaca mesin
- **Hak Menarik Persetujuan**: Menarik persetujuan kapan saja

## 5. Retensi Data
Kami menyimpan data sesuai periode berikut:
- Data profil: 7 tahun setelah penutupan akun
- Data leads: 5 tahun
- Data analitik: 2 tahun
- Persetujuan pemasaran: Hingga penarikan

## 6. Keamanan Data
Kami menerapkan:
- Enkripsi data saat transit dan istirahat
- Kontrol akses berbasis peran
- Audit keamanan rutin
- Protokol respons insiden

## 7. Pelanggaran Data
Jika terjadi pelanggaran data, kami akan:
- Memberitahu Anda dalam 3x24 jam
- Menjelaskan dampak dan langkah mitigasi
- Melaporkan ke otoritas yang berwenang

## 8. Transfer Data Internasional
Data Anda disimpan di server Indonesia. Transfer internasional hanya dilakukan dengan perlindungan yang setara.

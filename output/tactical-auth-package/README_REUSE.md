# Tactical Auth Kit - Lumina Style

Paket ini berisi komponen login dengan tema Tactical UI (Dark Mode, Emerald Accent) yang bisa digunakan kembali di proyek lain.

## Isi Paket
1. `LoginPage.tsx` - Komponen utama halaman login (Next.js App Router).
2. `auth-styles.css` - CSS khusus untuk perbaikan autofill dan animasi background.

## Cara Penggunaan di Proyek Baru

### 1. Prasyarat (Dependencies)
Pastikan proyek target sudah terinstall:
- `lucide-react` (untuk icon)
- `framer-motion` (opsional, jika ingin animasi lebih halus)
- `shadcn/ui` (Komponen Button, Input, Card)

### 2. Salin File
- Salin `LoginPage.tsx` ke folder `app/login/page.tsx` di proyek baru Anda.
- Salin isi `auth-styles.css` ke file `globals.css` proyek baru Anda.

### 3. Penyesuaian Path
Pastikan import komponen UI (`@/components/ui/...`) di `LoginPage.tsx` sudah sesuai dengan struktur folder di proyek baru Anda.

### 4. Mode Gelap
Pastikan proyek baru Anda mendukung mode gelap (Tailwind CSS `dark` class) agar tampilan maksimal.

# 🎯 Elite Hunter Dashboard - Setup Instructions

## 📦 Complete Installation Commands

```bash
# 1. Initialize Next.js project and install dependencies
npm install lucide-react clsx tailwind-merge recharts sqlite3

# 2. Initialize Tailwind CSS
npx tailwindcss init -p

# 3. Install and configure shadcn/ui components
npx shadcn-ui@latest init --defaults --dark

# 4. Install required shadcn/ui components
npx shadcn-ui@latest add button
npx shadcn-ui@latest add card
npx shadcn-ui@latest add avatar
npx shadcn-ui@latest add input
npx shadcn-ui@latest add dropdown-menu
npx shadcn-ui@latest add table
npx shadcn-ui@latest add badge
npx shadcn-ui@latest add dialog
npx shadcn-ui@latest add separator
npx shadcn-ui@latest add sheet
npx shadcn-ui@latest add tooltip

# 5. Start development server
npm run dev
```

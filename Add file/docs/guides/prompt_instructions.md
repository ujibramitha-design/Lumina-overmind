## 🔍 **Analysis Guidelines**

### **Lead Analysis Protocol**
```
[LEAD_ANALYSIS]
1. Extract key information:
   - Contact details (phone, email)
   - Property requirements (type, location, budget)
   - Urgency indicators (segera, butuh, cari)
   - Financial capability (price mentions, KPR keywords)

2. Score lead quality:
   - High (8-10): Complete info, high urgency, budget clear
   - Medium (4-7): Partial info, moderate urgency
   - Low (1-3): Minimal info, low urgency

3. Categorize intent:
   - Transactional: Beli, cari, butuh, survey, nego
   - Pain-Point: Khawatir, sulit, bingung, masalah
   - Comparison: Bandingkan, pilih, mana, lebih
   - Informational: Tanya, info, detail, spek

4. Generate recommendations:
   - Immediate action required
   - Follow-up schedule
   - Priority level
   - Custom message template
[/LEAD_ANALYSIS]
```

## 🎨 **Content Generation Guidelines**

### **Property Content Structure**
```
[PROPERTY_CONTENT]
1. Headline:
   - Location + Type + Key Feature
   - Example: "Rumah 2 Lantai di Cipocok Jaya - Dekat Pabrik"
   - Max 60 characters, include keywords

2. Opening Hook:
   - Main benefit or unique selling point
   - Address target audience pain point
   - Include urgency if applicable

3. Key Features (3-5 points):
   - Property specifications (type, luas, kamar)
   - Location advantages (akses, fasilitas)
   - Price benefits (promo, KPR, cicilan)
   - Quality highlights (material, sertifikat)

4. Call to Action:
   - Clear next step (hubungi, survey, booking)
   - Contact information
   - Urgency/scarcity element

5. Hashtags:
   - Location-based: #rumahserang #cipocokjaya
   - Feature-based: #rumahmurah #kprready
   - Benefit-based: #investasiproperti
[/PROPERTY_CONTENT]
```
